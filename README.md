# Felix UI

> VueJs UI components library based on Tailwind CSS

## Quick Install

```
npm install felix-ui
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run tests
```
npm run test:unit
```

### Watch tests
```
npm run test:watch
```
